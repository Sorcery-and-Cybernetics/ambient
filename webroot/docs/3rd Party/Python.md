## Installation
```
Check version: python --version
Check installed packages: pip list

Uninstall all packages: 
pip freeze > packages.txt
pip uninstall -y -r packages.txt

Uninstall Python from windows appstore, and regular add & remove programs (appwiz.cpl)

Install Python
In windows: Deactivate App execution aliases for python
Download installer from python website
Run installer as administrator

```
